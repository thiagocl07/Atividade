Desenvolva um sistema para gestão de estoque de uma papelaria escolar. O sistema deve permitir o
cadastro de produtos, categorias, fornecedores, entradas e saídas de estoque. Cada produto deve
possuir código, nome, categoria, quantidade atual, estoque mínimo e preço. O sistema deve permitir
registrar entrada, saída, ajuste de estoque, consulta de produtos abaixo do mínimo e histórico de
movimentações. Também deve impedir saídas com quantidade superior ao saldo disponível. A solução
deve respeitar toda a estrutura de camadas definida no PRD, com repositórios em memória e interfaces
separadas.
